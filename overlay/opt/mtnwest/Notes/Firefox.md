Firefox

Networking
2. Update DNS
    Right click on network connection.
    Click on Network Manager
    Click on Gear icon
    Go to DNS
    Add 9.9.9.9 to Primary
    Add 1.1.1.3 & 1.0.0.3 to Secondary
